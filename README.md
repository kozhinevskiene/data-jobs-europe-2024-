# Data Jobs in Europe – 2024

Project analyses **data jobs** in 23 European countries using 2024 job posting data. Focus is on the **Lithuanian market**.

## Tools

### Microsoft Power BI
- Data modeling (STAR schema)  
- Hierarchies  
- Dashboards  
- Drillthrough  
- Bookmarks  
- KPIs with implicit measures  

### Power Query
- Change data types  
- Fix missing values  
- Remove extra columns  
- Split and copy tables to make decimal tables and fact tables  

### DAX
- Create DAX measures  
- KPIs with explicit measures  
- Advanced DAX (Time Intelligence)  

## Data Source

Public dataset obtained from a YouTube tutorial (shared in the video comments).
